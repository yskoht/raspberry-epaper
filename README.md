# Raspberry e-paper

## Install

```sh
pip install raspberry-epaper
```

## Usage

```sh
epaper-print --device="model" picture.png
```

```sh
epaper-print --device="model" directory
```

```sh
epaper-print --device="model" sentence.txt
```

```sh
epaper-print --device="model" --qr="info" picture.png
```
